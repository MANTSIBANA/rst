test_fitacf

Syntax
test_fitacf --help
test_fitacf -old [options] datname
Synopsis

Searches for a record in a rawacf file and tests if it can be processed into fitacf.

Options
--helpprint the help message and exit.
--optionprint the options exit.
-vbverbose. Log information to the console.
-old use dat format.
-hr hrhour to search for.
-min mnminute to search for.
-sec scsecond to search for.
-beam bmbeam to search for.
Description

Returns zero if the record can be processed into fitacf data. Returns

Returns EOF if no record is found

Returns -1 for various other errors

Examples
test_fitacf -hr 10 -min 15 -sec 45 -beam 8 20150103.1000.02.bks.rawacf

This will search for a record in the 20150103.1000.02.bks.rawacf file that has an hour of 10, minute of 15, second of 45 and beam number of 8
